# SoftBoot
WordPress Theme

**SoftBoot Theme** for WordPress 

Required plugins:

 - Advanced Custom Fields  
 - Lightbox Plus Colorbox 

Additional plugins:
 - Advanced Page Manager
 - Better Delete Revision
 - Odnawianie miniaturek
 - WP Super Cache
 - All In One SEO Pack
 
Functionality:

  - 4 col. widget footer
  - Preloader
  - WordPress tile gallery
  - Fade In DOM Elements
